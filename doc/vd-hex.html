vd-hex Grid

Interactive hex-grid canvas component with selection, pan/zoom, terrain helpers, and optional grid rotation. Built on flat-top axial coordinates — suitable for visualization, prototyping, and experimental game mechanics.

Usage Example 

<div id="hex-demo-container">
    <canvas id="hex-demo"></canvas>
</div>
import { VdHexGrid } from './hex-grid.js';

const grid = new VdHexGrid({
    element: document.getElementById('hex-demo-container'),
    canvas: document.getElementById('hex-demo'),
    size: 30,
    width: 15,
    height: 10,
    rotation: 0
});

grid.on('select', (hex) => {
    console.log('Selected hex:', hex.q, hex.r);
});

grid.setHexTerrain(0, 0, 'GRASSLAND');
grid.setHexData(0, 0, { owner: 'blue', label: 'Spawn' });

grid.setSize(40);
grid.setDimensions(20, 15);
grid.setRotation(-Math.PI / 6);

Hex Coordinate System

VdHexGrid uses axial coordinates (q, r) with flat-top orientation:

  • q — Column coordinate (horizontal axis)
  • r — Row coordinate (diagonal axis)
Flat-Top Hexagon Layout:

        r
        ↑
        │   ╱ ╲
        │  ╱   ╲
        │ ╱     ╲
        │╱       ╲
        ┼─────────→ q
        │╲       ╱
        │ ╲     ╱
        │  ╲   ╱
        │   ╲ ╱

Adjacent Hexes (6 directions):
  East:       (q+1, r)
  Northeast:  (q+1, r-1)
  Northwest:  (q, r-1)
  West:       (q-1, r)
  Southwest:  (q-1, r+1)
  Southeast:  (q, r+1)

Distance formula: (|q1-q2| + |q1+r1-q2-r2| + |r1-r2|) / 2
